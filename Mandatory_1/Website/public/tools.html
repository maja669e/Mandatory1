Visual Studio Code

Visual Studio Code is an editor, where you can write code in different languages.
We have been using it for writing and running our Node.js code.

Nodemon

Nodemon is a helping tool for running node.js applications. It runs automatically, and will update your application everytime you save your project. Instead of doing this manually with the command "node name.js", it will simply keep updating on it's own.

REST

Rest has been used to send different HTTP transactions and HTTP methods.
In the example you can see that we have used the methods post and get, which allows us to get and create objects.
